## 🏠 홈 화면

- Firebase 연결 상태 표시
- 라이딩 이름 입력 후 방 만들기 (최대 10자)
- 진행 중인 라이딩 목록 실시간 표시
  - 참여 인원 · 생성 시간 · 선두 속도 표시
  - 내 방: 👑 표시 + 🗑 종료 버튼
  - 다른 방: 입장 버튼
- 빈 방 / name 없는 유령 방 자동 제외 및 삭제

## 🗺️ 라이딩 방

### 상단 HUD
```
[← 나가기] [방 이름] [🗺️] [👥 N명] [🔗]
```
| 버튼 | 기능 |
|------|------|
| ← 나가기 | 방 나가기 (주황색) |
| 🗺️ | GPX 코스 등록/관리 |
| 👥 N명 | 팀원 현황 패널 토글 |
| 🔗 | 초대 링크 복사 |

### 지도
- OpenStreetMap 기반 (줌 레벨 18)
- 팀원 위치 실시간 마커 (이름 2글자 + 하단 이름 라벨)
- 밝기 자동 감지 → 밝은 색상엔 검정 글자, 어두운 색상엔 흰 글자
- 내 마커: 크게 + 흰색 링 강조 / 방장: 👑 표시
- GPS 20초 이상 미수신 시 🔴 표시

### 우측 버튼
| 버튼 | 기능 |
|------|------|
| 🆘 | 긴급 도움 요청 (빨간색 맥박 애니메이션) |
| 📊 | 코스 정보 (GPX 등록 시만 표시) |
| 📍 | 내 위치로 이동 + 자동 추적 ON |
| 👥 | 전체 팀원 지도에 맞추기 |
| ⚡ | 속도계 표시 (10초 후 자동 숨김) |

### 자동 추적 (followMe)
- 첫 GPS 수신 시 자동 추적 ON → 내 위치 항상 지도 중앙
- 지도 드래그 시 추적 OFF
- 팀원 탭 시 해당 위치로 이동 (추적 OFF)

---

## 👥 팀원 현황 패널

- 좌측 하단 고정 · 👥 버튼으로 접기/펼치기
- **팀원 현황 · N명** 인원수 실시간 표시
- 인원수에 따라 패널 높이 자동 조정 (최대 620px, 10명 기준)
- GPX 있을 때: 경로상 진행 거리 기준 정렬
- GPX 없을 때: 속도 기준 정렬
- 표시 정보: 이름 · 속도(km/h) · 나와의 거리
- 거리 색상: 🟢 300m 이내 / 🟡 300~1000m / 🔴 1km 초과
- **팀원 탭 → 해당 위치로 지도 이동**

---

## 👑 방장 관리

- 방 생성자가 방장
- 방장 나가면 다른 팀원에게 자동 이관
- 마지막 인원 나가면 방 자동 삭제
- 방장만 GPX 등록/삭제 가능
- 새 방장 지정 시 토스트 알림

---

## 🗺️ GPX 코스

- 방 입장 후 🗺️ 버튼으로 GPX 파일 업로드 (방장만)
- 지도에 파란 실선으로 코스 표시
- Firebase 저장 → 팀원 전체 자동 공유
- **현재 GPX는 1회용** (방 종료 시 함께 삭제)

### 코스 포인트 마커
| 마커 | 위치 |
|------|------|
| 🚦 출발점 | 경로 시작 |
| 🏁 도착점 | 경로 끝 |
| 🔄 반환점 | 경로 50% 지점 |
| 📍 주요 포인트 | 25%, 75% 지점 |

### 경로 이탈 알림
| 모드 | 동작 |
|------|------|
| 🔕 없음 | 알림 없음 |
| 🔔 나만 | 진동 + 소리 + 배너 |
| 📢 전체 | 진동 + 소리 + 배너 + 전체 팀원 배너 |

- 코스 50m 이상 이탈 시 발동
- 배너 10초 후 자동 사라짐 / 60초 반복 억제
- 라이딩 중 모드 변경 가능

---

## 📊 코스 정보 (GPX 등록 시)

📊 버튼 탭 → 하단 모달로 표시

### 항상 표시
- 총 거리 (km) · 총 포인트 수
- 주요 포인트별 출발점부터의 거리
- 포인트 탭 → 해당 위치로 지도 이동

### 고도(ele) 데이터 있을 때 추가
- 총 상승고도 / 총 하강고도
- 경사도 분포 (완만 0~3% / 보통 3~7% / 급경사 7%+)

> Garmin Connect, Strava 등에서 내보낸 GPX는 고도 포함  
> 카카오맵, 네이버 지도 GPX는 고도 미포함

- 선두와의 경로 거리 · 예상 도달 시간 · 속도 차이

## 🆘 긴급 도움 요청 (SOS)

- 우측 **[🆘]** 버튼 (빨간색 맥박 애니메이션)
- 클릭 → "팀원에게 긴급 도움을 요청할까요?" 확인

### 발신자
- Firebase에 SOS 이벤트 저장 (uid별 독립 저장)
- 여러 명 동시 SOS 가능

### 수신자 (팀원 화면)
- 화면 정중앙에 **정사각형 카드** 팝업
```
┌─────────────────────────┐
│          🆘              │
│       긴급 도움 요청     │
│   지훈님이 도움을 요청!  │
├─────────────────────────┤
│  [📍 위치 확인]          │
└─────────────────────────┘
```
- **진동 + 경보음** 즉시 발생
- **10초마다 반복** (최대 10분 = 60회)
- [📍 위치 확인] 클릭 → 배너 닫힘 + 진동/소리 중단 + SOS 위치로 지도 이동
- 여러 명 SOS 시 이름 모두 표시

## ⏱️ 라이딩 방 생명주기

| 조건 | 동작 |
|------|------|
| 멤버 0명 | 방 즉시 자동 삭제 |
| 1시간 무활동 (속도 0) | 55분 경과 시 경고 → 60분 후 자동 종료 |
| GPX 도착점 100m 이내 전원 도달 | "🎉 모든 팀원이 도착했어요!" 알림 (자동 종료 없음) |
| 방장 직접 나가기 | 다른 팀원에게 방장 이관 또는 방 삭제 |
| 로그아웃 | 방 정리 후 로그인 화면 |

- **lastActivity** 기준 1시간 초과 방 → 목록에서 숨김
- 탭 전환 / 새로고침 시 방 유지 (로그아웃 시에만 정리)

## 🏗️ 기술 구조

```
Firebase Realtime Database
└── rooms/{roomId}/
    ├── id, name, ownerId, createdAt, lastActivity
    ├── members/{userId} → name, color, lat, lng, speed, ts
    ├── gpx[]            → GPX 포인트 배열
    ├── gpxAlertMode     → 0/1/2 (없음/나만/전체)
    ├── gpxDeviation     → 이탈 이벤트 (uid, name, ts)
    └── sos/{userId}     → uid, name, lat, lng, ts

localStorage (WE_ prefix)
├── WE_user        → id, name
├── WE_ci          → 색상 인덱스
└── WE_fbProjectId → Firebase 프로젝트 ID
```
