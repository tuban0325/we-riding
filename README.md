**배포 주소:** https://tuban0325.github.io/we-riding/  
**기술 스택:** HTML/CSS/JavaScript · Firebase Realtime Database · Leaflet · OpenStreetMap

- 사용자별 고유 색상 자동 부여
- 우측 상단 이름 버튼 → 로그아웃

---

## 🏠 홈 화면

- Firebase 연결 상태 표시
- 라이딩 이름 입력 후 방 만들기
- 진행 중인 라이딩 목록 실시간 표시 (참여 인원 · 생성 시간)
- 내 방: 👑 표시 + 🗑 종료 버튼 / 다른 방: 입장 버튼
- 멤버 0명인 빈 방 자동 제외

---

## 🗺️ 라이딩 방

### 상단 버튼
| 버튼 | 기능 |
|------|------|
| ← | 방 나가기 |
| 🗺️ | GPX 코스 등록/관리 |
| 👥 N명 | 팀원 현황 패널 토글 |
| 🔗 | 초대 링크 복사 |

### 지도
- OpenStreetMap 기반
- 팀원 위치 실시간 마커 표시 (이름 2글자)
- 내 마커: 크게 + 흰색 테두리 / 방장: 👑 표시
- GPS 20초 이상 미수신 시 🔴 표시

### 우측 버튼
| 버튼 | 기능 |
|------|------|
| 📊 | 코스 정보 (GPX 등록 시만 표시) |
| 📍 | 내 위치로 이동 + 자동 추적 ON |
| 👥 | 전체 팀원 지도에 맞추기 |
| ⚡ | 속도계 표시 (10초 후 자동 숨김) |

### 자동 추적
- 첫 GPS 수신 시 자동 추적 ON → 내 위치 항상 지도 중앙
- 지도 드래그 시 추적 OFF
- 팀원 탭 시 해당 위치로 이동 (추적 OFF)

---

## 👥 팀원 현황 패널

- 좌측 하단 고정, 👥 버튼으로 접기/펼치기
- GPX 있을 때: 경로상 진행 거리 기준 정렬
- GPX 없을 때: 속도 기준 정렬
- 표시 정보: 이름 · 속도(km/h) · 나와의 거리
- 거리 색상: 🟢 300m 이내 / 🟡 300~1000m / 🔴 1km 초과
- 팀원 탭 → 해당 위치로 지도 이동

---

## 👑 방장

- 방 생성자가 방장
- 방장 나가면 다른 팀원에게 자동 이관
- 마지막 인원 나가면 방 자동 삭제
- 방장만 GPX 등록/삭제 가능
- 새 방장 지정 시 토스트 알림

---

## 🗺️ GPX 코스

- 방 입장 후 🗺️ 버튼으로 GPX 파일 업로드
- 지도에 파란 실선으로 코스 표시
- Firebase 저장 → 팀원 전체 자동 공유
- **현재 GPX는 1회용** (방 종료 시 함께 삭제)

### 코스 포인트 마커
| 마커 | 위치 |
|------|------|
| 🚦 출발점 | 경로 시작 |
| 🏁 도착점 | 경로 끝 |
| 🔄 반환점 | 경로 50% 지점 |
| 📍 주요 포인트 | 25%, 75% 지점 |

### 경로 이탈 알림
| 모드 | 동작 |
|------|------|
| 🔕 없음 | 알림 없음 |
| 🔔 나만 | 진동 + 소리 + 배너 |
| 📢 전체 | 진동 + 소리 + 배너 + 전체 팀원 배너 |

- 코스 50m 이상 이탈 시 발동
- 배너 10초 후 자동 사라짐 / 60초 반복 억제
- 라이딩 중 모드 변경 가능

---

## 📊 코스 정보 (GPX 등록 시)

📊 버튼 탭 → 하단 모달로 표시

### 항상 표시
- 총 거리 (km)
- 총 포인트 수
- 주요 포인트별 출발점부터의 거리
- 포인트 탭 → 해당 위치로 지도 이동

### 고도(ele) 데이터 있을 때 추가
- 총 상승고도 / 총 하강고도
- 경사도 분포 (완만 0~3% / 보통 3~7% / 급경사 7%+)

> Garmin Connect, Strava 등에서 내보낸 GPX는 고도 포함  
> 카카오맵, 네이버 지도 GPX는 고도 미포함

## 🏗️ 기술 구조

```
Firebase Realtime Database
└── rooms/{roomId}/
    ├── id, name, ownerId, createdAt
    ├── members/{userId} → name, color, lat, lng, speed, ts
    ├── gpx[]            → GPX 포인트 배열 (방 종료 시 삭제)
    ├── gpxAlertMode     → 0/1/2
    └── gpxDeviation     → 이탈 이벤트 (uid, name, ts)

localStorage (WE_ prefix)
├── WE_user        → id, name
├── WE_ci          → 색상 인덱스
└── WE_fbProjectId → Firebase 프로젝트 ID
```
