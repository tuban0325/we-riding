## 👤 로그인

- 닉네임 입력만으로 시작 (회원가입 불필요)
- 재접속 시 이름 자동 입력 (localStorage 저장)
- Firebase 자동 연결 (설정 화면 없음)
- 사용자별 고유 색상 자동 부여
- 우측 상단 **[⏻ 로그아웃]** 버튼 (빨간색 고정)

### 중복 이름 처리
- 로그인 시 전체 방에서 동일 이름 사용자 확인
- **같은 IP or 연결 끊김** → 동일 사용자 재접속으로 판단 → 자동 전환
- **다른 IP + 활성 중** → "이름 사용 중" 안내 → 다른 이름 유도
- 다른 기기에서 같은 uid로 접속 시 기존 세션 자동 강제 로그아웃

## 🏠 홈 화면

- Firebase 연결 상태 표시
- 라이딩 이름 입력 후 방 만들기 (최대 10자)
- 진행 중인 라이딩 목록 실시간 표시
  - 참여 인원 · 생성 시간 · 선두 속도 표시
  - 내 방: 👑 표시 + 🗑 종료 버튼
  - 다른 방: 입장 버튼
- 빈 방 / 유령 방 자동 제외 및 삭제
- lastActivity 1시간 초과 방 목록에서 숨김

## 🗺️ 라이딩 방

### 상단 HUD
| 버튼 | 기능 |
|------|------|
| ← 나가기 | 방 나가기 (주황색) |
| 🗺️ | GPX 코스 등록/관리 (방장만) |
| 👥 N명 | 팀원 현황 패널 토글 |
| 🔗 | 초대 링크 복사 |

### 지도
- OpenStreetMap 기반 (줌 레벨 16)
- 팀원 위치 실시간 마커 (이름 라벨 + 꼬리)
- 밝기 자동 감지 → 밝은 색 검정 글자 / 어두운 색 흰 글자
- 내 마커: 크게 + 흰색 링 강조 / 방장: 👑 표시
- GPS 20초 이상 미수신 시 🔴 표시

### 우측 버튼 (위 → 아래)
| 버튼 | 상태 | 기능 |
|------|------|------|
| 🆘 | 빨간 맥박 | 긴급 도움 요청 |
| 📊 | GPX 등록 시만 | 코스 정보 |
| 📍 | 주황=추적중 | 내 위치 이동 + 추적 ON |
| 👥 | 주황=전체보기 | 전체 팀원 지도 맞추기 |
| ⚡ | 주황=켜짐 | 속도계 토글 (다시 누를 때까지 유지) |

### 버튼 상태 표시
- **📍** 내 위치 추적 중 → 주황색 활성 / 드래그·팀원 탭 시 비활성
- **👥** 전체보기 → 클릭 후 1초 주황 활성
- **👥 상단** 팀원 패널 보임 → 주황 활성 / 숨김 → 비활성
- **⚡** 다시 누를 때까지 유지 (라디오 버튼 방식)

### 자동 추적 (followMe)
- 첫 GPS 수신 시 자동 추적 ON
- 지도 드래그 시 추적 OFF
- 팀원 탭 시 해당 위치로 이동 (추적 OFF)

## 👥 팀원 현황 패널

- 좌측 하단 고정 · 👥 버튼으로 접기/펼치기
- **팀원 현황 · N명** 인원수 실시간 표시
- 인원수에 따라 패널 높이 자동 조정 (최대 620px)
- GPX 있을 때: 경로 진행 거리 기준 정렬
- GPX 없을 때: 속도 기준 정렬
- 거리 색상: 🟢 300m 이내 / 🟡 300~1000m / 🔴 1km 초과
- 팀원 탭 → 해당 위치로 지도 이동 (줌 16)

---

## 👑 방장 관리

- 방 생성자가 방장
- 방장 나가면 다른 팀원에게 자동 이관
- 마지막 인원 나가면 방 자동 삭제
- **방장만:** GPX 등록/삭제, 경로 이탈 알림 모드 변경 가능
- 팀원이 변경 시도 시 "🔒 방장만 변경할 수 있습니다" 표시

---

## 🗺️ GPX 코스

- 🗺️ 버튼 → GPX 파일 업로드 (방장만)
- 지도에 파란 실선으로 코스 표시
- Firebase 저장 → 팀원 전체 자동 공유
- **현재 GPX는 1회용** (방 종료 시 함께 삭제)

### 코스 포인트 마커
| 마커 | 위치 |
|------|------|
| 🚦 출발점 | 경로 시작 |
| 🏁 도착점 | 경로 끝 |
| 🔄 반환점 | 경로 50% 지점 |
| 📍 주요 포인트 | 25%, 75% 지점 |

### 경로 이탈 알림 (방장만 변경)
| 모드 | 동작 |
|------|------|
| 🔕 없음 | 알림 없음 |
| 🔔 나만 | 진동(400ms×4) + 소리(1318Hz) + 주황 화면 점멸 + 배너 |
| 📢 전체 | 위 동일 + 전체 팀원 배너 |

- 코스 50m 이상 이탈 시 발동 / 60초 반복 억제

## 🆘 긴급 도움 요청 (SOS)

### 발신
- 우측 [🆘] 버튼 → "팀원에게 긴급 도움을 요청할까요?" 확인
- Firebase에 uid별 독립 저장 (여러 명 동시 SOS 가능)

### 수신 알림 (3중 알림)
```
1. 화면 전체 빨간 점멸 🔴 (8회, 저지 주머니에서도 빛 새어나옴)
2. 진동 500ms×4 (강하고 길게)
3. 경보음 1760~2640Hz 최대 볼륨 (높은 음역대)
```

- 10초마다 반복 (최대 10분 = 60회)
- 화면 정중앙 빨간 카드 팝업
- [📍 위치 확인] → 배너 닫힘 + 진동/소리 중단 + SOS 위치로 지도 이동

### 백그라운드 수신 (화면 닫힘)
- Service Worker가 10초마다 Firebase REST 폴링
- 스마트폰 상단 푸시 알림 표시

| 환경 | 지원 |
|------|------|
| Android Chrome | ✅ |
| iOS 16.4+ Safari | ✅ |
| 카카오 인앱 브라우저 | ⚠️ 제한적 |

## 🏗️ 기술 구조

### Firebase Realtime Database
```
rooms/{roomId}/
├── id, name, ownerId, createdAt, lastActivity
├── members/{userId}
│   ├── name, color, lat, lng, speed, ts
│   ├── sessionKey  ← 세션 관리
│   └── ip          ← 중복 접속 판단
├── gpx[]            → GPX 포인트 배열
├── gpxAlertMode     → 0/1/2
├── gpxDeviation     → 이탈 이벤트
└── sos/{userId}     → uid, name, lat, lng, ts
```

### localStorage (WE_ prefix)
```
WE_user        → id, name
WE_ci          → 색상 인덱스
WE_fbProjectId → Firebase 프로젝트 ID
WE_sessionKey  → 세션키 (중복 로그인 방지)
WE_cachedIp    → IP 캐시 (중복 이름 판단)
```

### JS 섹션 구분 (index.html)
```
SECTION 1:  Firebase 초기화
SECTION 2:  로그인 / 초기화
SECTION 3:  홈 화면 / 방 목록
SECTION 4:  GPX 기능
SECTION 5:  방 관리 (만들기/입장/나가기)
SECTION 6:  지도 / GPS / Firebase 동기화
SECTION 7:  팀원 패널 / 선두 카드 / 알림
SECTION 8:  방 생명주기 / SOS / 코스 정보
SECTION 9:  공유 / 로그아웃 / 기타
SECTION 10: PWA / Service Worker / 푸시 알림
```
